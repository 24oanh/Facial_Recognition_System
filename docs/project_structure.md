## Mục đích các thư mục trong dự án PCA + KNN vs PCA + SVM

- **data/**
  - **raw/**: Dữ liệu gốc ORL/AT&T.
  - **processed/**: Dữ liệu đã tiền xử lý hoặc cache trung gian nếu cần.
- **notebooks/**
  - Notebook dùng để khám phá dữ liệu, thử công thức và demo nhanh.
- **report/**
  - Tài liệu báo cáo cuối kỳ, hình minh họa riêng cho báo cáo.
- **results/**
  - **figures/**: Biểu đồ, confusion matrix, eigenfaces.
  - **metrics/**: Kết quả đánh giá dạng `.csv` hoặc `.json`.
- **src/**
  - **datasets/**: Định nghĩa container dataset như `FaceDataset`.
  - **preprocessing/**:
    - `loader.py`: đọc ảnh ORL/AT&T, chia train/test.
    - `preprocessor.py`: resize, grayscale, normalize, flatten.
  - **features/**:
    - `pca.py`: triển khai PCA thuần NumPy.
    - `feature_extractor.py`: wrapper dùng PCA như bộ trích xuất đặc trưng.
  - **models/**:
    - `knn.py`: KNN thuần NumPy.
    - `svm.py`: wrapper SVM dùng cho pipeline thực nghiệm.
    - `pca.py`, `pca_knn.py`, `pca_svm.py`: wrapper tương thích ngược cho import cũ.
  - **pipelines/**:
    - `pca_knn.py`: pipeline `PCA -> KNN`.
    - `pca_svm.py`: pipeline `PCA -> SVM`.
    - `train_pipeline.py`: helper train.
    - `eval_pipeline.py`: helper evaluate.
  - **experiments/**: Script chạy quét siêu tham số và so sánh mô hình.
  - **configs/**: Cấu hình dữ liệu, tham số thí nghiệm, đường dẫn output.
  - **utils/**: Metrics và trực quan hóa.
- **webapp/**
  - **saved_models/**: Nơi lưu model artifact để phục vụ demo.
  - `app.py`: ứng dụng demo nếu cần.
- **docs/**
  - Tài liệu quản lý đồ án và mô tả cấu trúc dự án.
